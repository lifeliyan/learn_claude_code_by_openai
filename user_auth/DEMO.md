# 用户登录系统演示指南

## 快速演示

### 1. 启动系统
```bash
cd user_auth
./start.sh
```

### 2. 访问应用
打开浏览器访问：http://localhost:5000

### 3. 使用演示账户
- **用户名**: admin
- **密码**: admin123

## 功能演示步骤

### 步骤1：首页浏览
1. 访问 http://localhost:5000
2. 查看系统介绍和特性
3. 观察导航栏（显示登录/注册链接）

### 步骤2：用户注册
1. 点击"注册新账户"
2. 填写注册信息：
   - 用户名：testuser
   - 邮箱：test@example.com
   - 密码：Test123!
   - 确认密码：Test123!
3. 观察密码强度指示器
4. 点击"注册账户"
5. 查看成功提示

### 步骤3：用户登录
1. 点击"立即登录"
2. 使用刚注册的账户登录：
   - 用户名：testuser
   - 密码：Test123!
3. 可选：勾选"记住我"
4. 点击"登录"
5. 查看欢迎消息

### 步骤4：个人资料
1. 登录后点击右上角用户名
2. 查看个人资料页面
3. 观察用户信息：
   - 用户ID
   - 注册时间
   - 最后登录时间
   - 账户状态

### 步骤5：安全功能测试
1. **登录保护测试**：
   - 登出账户
   - 尝试使用错误密码登录5次
   - 观察第6次尝试时的错误提示

2. **会话管理测试**：
   - 登录时勾选"记住我"
   - 关闭浏览器
   - 重新打开浏览器访问应用
   - 观察是否保持登录状态

### 步骤6：API接口测试
1. 打开浏览器开发者工具
2. 访问：http://localhost:5000/api/check_auth
3. 查看返回的JSON数据
4. 登录前后对比响应内容

## 故障排除

### 常见问题
1. **端口占用**：
   ```bash
   # 查找占用5000端口的进程
   lsof -i :5000
   
   # 终止进程
   kill <PID>
   ```

2. **依赖问题**：
   ```bash
   # 重新安装依赖
   pip install -r requirements.txt --force-reinstall
   ```

3. **数据库问题**：
   ```bash
   # 删除并重建数据库
   rm users.db
   python app.py
   ```
