安全信息

密码安全

您的密码使用bcrypt算法进行哈希存储，确保即使数据库泄露也不会暴露明文密码。

登录保护

系统会记录所有登录尝试，防止暴力破解攻击。连续失败将暂时锁定账户。

会话安全

使用安全的HTTP-only Cookie存储会话信息，防止XSS攻击窃取会话。

数据加密

敏感信息在传输和存储过程中都进行了适当的加密处理。

API访问

您的账户可以通过以下API接口进行访问：

// 检查认证状态
fetch('/api/check_auth')
    .then(response => response.json())
    .then(data => {
        if (data.authenticated) {
            console.log('当前用户：', data.user.username);
        } else {
            console.log('未登录');
        }
    });
